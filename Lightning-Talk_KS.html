R Markdown

  • a framework for creating fully reproducible dynamic documents in R in multiple formats such as
    • Word
    • PDF
    • HTML

Installation

  • Install from CRAN with:
install.packages("rmarkdown")

Distill R Markdown

  • An R Markdown template designed for scientific papers
  • Best suited for technical and scientific presentations
  • Functionalities to insert:
    • author information
    • text
    • tables
    • figures
    • references
    • mathematical symbols or equations (with LaTeX implementation)
    • etc.

Get Started with Distill

Installation:

  • Install the released version of distill from CRAN:
install.packages("distill")
  • Or install the development version from GitHub with:
install.packages("devtools")
devtools::install_github("rstudio/distill")

Loading package:

library(distill)

Distill .Rmd file components (1)

  • YAML (Header)
    • Authors info
    • Output formats
    • Style settings
    • Adding a BibTeX file for references
    • Creating automatic citation boxes

Distill .Rmd file components (2)

  • Content (after the r setup code chunk - given by default)
    • Where we actually start writing!
    • Same writing format as the usual R Markdown :)
    • LaTeX friendly space ~

YAML (Header) - cont.1

  • Output Formats
output:
  html: distill::distill_article
  pdf_document:
    template: null
  html_document:
    df_print: paged
  word_document:
    reference_docx: refs/word-styles-reference-01.docx

YAML (Header) - cont.2

  • Styles, References, Citation
graphics: yes
font-family: Helvetica
images: path/to/images
bibliography: refs/connectivity_score_review.bib (path/to/bibtexfile)
csl: refs/briefings-in-bioinformatics.csl (path/to/cslfile)
biblio-style: apa
---
